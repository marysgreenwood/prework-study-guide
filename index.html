Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements within the body help organize content for a better user experience.
  • Semantic elements help organize the webpage and make code more legible for other developers.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A declaration contains
    • a selector (refers to a class, id, element, or element type in the html file)
    • an attribute (e.g. color, margin etc.)
    • a value (e.g pink, 40 px, etc.)

Git

  • git status tells you which branch you are working on and how it differs from the main branch
  • git add -A (the capital A is important apparently) stages edits for commit
  • git commit -m "message" commits changes with message describing what has been altered
  • git push uploads latest commit to github
